Redis 简介

、{{ $image := .Resources.Get “image.png” }} 2 Redis（Remote Dictionary Server）是一个开源的内存数据结构存储系统，广泛应用于缓存、消息队列、实时分析等场景。它支持多种数据结构，如字符串、哈希、列表、集合、有序集合、位图、HyperLogLog 和地理空间索引等。

特点

  • 高性能：Redis 通过内存存储和高效的数据结构操作，提供了极高的读写性能。
  • 丰富的数据类型：支持多种数据结构，满足不同应用场景的需求。
  • 持久化：支持 RDB 和 AOF 两种持久化方式，确保数据的持久性。
  • 高可用性：通过主从复制、哨兵模式和集群模式，实现高可用性和数据分片。
  • Lua 脚本：支持 Lua 脚本，提供了更强大的数据操作能力。
  • 事务：支持事务操作，保证一组命令的原子性执行。

安装

在 Linux 上安装

  1. 下载 Redis 源码：

    1

    wget http://download.redis.io/releases/redis-6.2.6.tar.gz

  2. 解压并编译：

    1
2
3

    tar xzf redis-6.2.6.tar.gz
cd redis-6.2.6
make

  3. 启动 Redis 服务器：

    1

    src/redis-server

在 Windows 上安装

可以使用第三方提供的 Windows 版本 Redis，例如 Memurai 或 Redis for Windows。

基本命令

字符串

  • 设置键值对：

    1

    SET key value

  • 获取键值：

    1

    GET key

哈希

  • 设置哈希字段：

    1

    HSET myhash field1 value1

  • 获取哈希字段：

    1

    HGET myhash field1

列表

  • 在列表头部插入元素：

    1

    LPUSH mylist value

  • 获取列表中的元素：

    1

    LRANGE mylist 0 -1

集合

  • 添加元素到集合：

    1

    SADD myset value

  • 获取集合中的所有元素：

    1

    SMEMBERS myset

有序集合

  • 添加元素到有序集合：

    1

    ZADD myzset score value

  • 获取有序集合中的元素：

    1

    ZRANGE myzset 0 -1 WITHSCORES
